# Memory Card

A memory card game where players must avoid selecting the same card twice.

[Live Demo](https://memory-card-kappa-one.vercel.app/)

## Tech Stack

- [React](https://react.dev/)
- [Vite](https://vite.dev/)
- [PokeAPI](https://pokeapi.co/)

## Run Locally

```bash
npm install
npm run dev
```
